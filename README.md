# test

お問い合わせフォームと在庫発注点管理のウェブアプリ一式です。

## 含まれるもの

- **シンプル_お問い合わせ** … シンプルなお問い合わせフォーム
- **夢ふわ_お問い合わせ** … 夢ふわスタイルのお問い合わせフォーム
- **動物園_お問い合わせ** … 動物園テーマのお問い合わせフォーム
- **水族館_お問い合わせ** … 水族館テーマのお問い合わせフォーム
- **在庫発注点管理** … スプレッドシート連携の在庫発注点管理ウェブアプリ（GAS デプロイ対応）

## 使い方

各フォルダの `index.html` をブラウザで開くか、在庫発注点管理は GAS の Web アプリとしてデプロイして利用できます。
